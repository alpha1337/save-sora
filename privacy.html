Save Sora Privacy Policy

Effective date: April 13, 2026

Overview

Save Sora is a local browser extension that helps users fetch, review, and export their Sora library data. It does not operate a remote backend and does not sell personal data.

What Data Is Processed

The extension processes Sora metadata visible to the signed-in user in order to render the dashboard, filter results, generate CSV exports, and build ZIP archives.

Data is stored locally in the browser extension storage/IndexedDB for runtime use and export features.

Permissions and Why They Are Needed

Permission Purpose
downloads Download selected videos and generated archive files to the user's machine.
scripting Run fetch helpers in authenticated Sora/ChatGPT tabs initiated by the user.
storage Store local preferences and app state.
tabs Detect/reuse Sora tabs and coordinate worker tabs for queued processing.
unlimitedStorage Prevent local quota failures during large fetch sessions by allowing storage of high-volume normalized metadata rows, download history indexes, and export preparation state.

Data Sharing

Save Sora does not intentionally transmit your fetched library metadata to third-party analytics, ad networks, or data brokers.

Data Retention and Deletion

Stored data remains local to the browser profile until you clear extension data or uninstall the extension.
